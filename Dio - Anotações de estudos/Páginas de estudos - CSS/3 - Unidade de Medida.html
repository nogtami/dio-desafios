Unidades de Medida

Este documento contém anotações dos conteudos passados no curso de "Unidade de Medida", da Dio.me.

Unidades de medida absolutas

São tamanhos que não estão atrelados a nenhum outro valor de referencia. Ou seja, elas não dependem de nenhu outro valor para serem calculadas.
Exemplo: cm (centimetros), mm (milimetros), Q (quatro de milimetro), in (polegadas), pc (paica), pt (pontos), px (pixels), entre outras.

  • Estas unidades de medidades são fixas e não podem mudar de acordo com os dispositivos ou outro elemento.
  • O resultado será sempre o mesmo.
  • Usadas quando conhecemos as caracteristicas e configurações dos dispositivos ou quando não queremos que estas caracteristicas mudem por interferencia de outro fator ou elemento.
  • Podem ser mais uteis quando trabalhadas para impressão do que para o meio digital.
  • Pixels é a unidade de medida absoluta mais comum para meios digitais.
    • É considerada a menor unidade de alguma representação digital.
    • Esta unidade não possui um valor fixo e varia de acordo com a densidade da tela, que é calculada pela divisão da resolução da tela (quantidade de pixels) e o tamanho da tela (quantidade de polegadas). Assim, a unidade utilizada para definir a densidade de um tela é a ppi (pixels por polegada).
    • Pixels físicos X Pixels lógicos:
      • Pixels físicos são pixels reais, que existem na tela do dispositivo digital. (Cada "luzinha" da tela)
      • Pixels lógicos são o quanto um conteúdo cabem dentro da tela de um dispositivo, independente da sua densidade.
      • A resolução de uma tela é calculada com a quantidade de pixels físicos que ela possui em altura e largura (resolução = px.alt x px.larg).
        O cálculo de densidade numa tela é: densidade = (px.alt x px.larg)/ins.
      • Quanto maior a densidade de uma tela, maior será a definição de seu contéudo.
      • Entretanto, o CSS só está se importanto com a quantidade lógica de pixels (ou seja, o quanto de conteudo cabe na tela, independente de sua densidade). Ele não se preocupa com esta questão de densidade dos pixels.

Unidades de medida relativas

São unidades calculadas com base em unidades já conhecidas de um outro elemento, o que faz com que o resultado possa variar de acordo com o valor da propriedade do elemento no qual estamos nos baseando.

  • Pode ajudar a dimensionar um elemento em relação a outro.
    Exemplo: um texto em relação ao tamanho de uma tela ou outro elemento que possa ser seu container.
  • Auxiliam na construção de responsidade de um site ao ajudarem a adaptar os elementos corretamente aos dispositivos utilizados pelos usuários.
  • São calculos feitos pelo navegadores que traduzem o nosso conteudo em pixels nas telas.

Algumas unidades de medida relativas são:

  • porcentagem: é uma unidade de medida que tomará por referencia o elemento pai para se basear, podendo alterar o resultado de acordo com o lugar que está inserida.

  • em: é utilizada para definir o tamanho do elemento através da propriedade fonte-size.
    • Quando para a propriedade font-size, vai tomar como referencia o valor da propriedade font-size do elemento pai no qual o elemento filho está inserido.
    • Quando para outras propriedades que não a font-size, vai usar como referencia o valor do font-size do mesmo elemento.
    • Como uma unidade de medida que multiplica valores a partir da relação entre elementos pai-filho, precisa tomar cuidado para não perder controle das medidas.
    • Utilizar o em, assim como o rem, traz flexibilidade aos projetos e capacidade de dimensionar os elementos das páginas web, o que resulta em facildade de manitenção de codigo, designer responsivos e controle da escala dos navegadores.
    • Porque utilizar o em?
      • Escala definida pelo contexto no qual o elemento está inserido.

  • rem: é uma unidade que, assim como a em, utiliza o font-size para basear o tamanho do seus elementos. Entretanto, faz isso através de um elemento raiz, que no caso é a tag html.
    • Como padrão, esta tag tem como font-size o valor de 16 px.
    • Todos os valores utilizarem como medida o rem vão consultar o valor da propriedade font-size da tag raiz html para se basearem, independente da sua relação elemento pai-filho.
    • Utilizar o rem, assim como o em, traz flexibilidade aos projetos e capacidade de dimensionar os elementos das páginas web, o que resulta em facildade de manitenção de codigo, designer responsivos e controle da escala dos navegadores.
    • Porque utilizar o rem?
      • Tamanhos de unidades consistentes, sem herança.
      • Melhor experiência da pessoa usuária.
      • Facilita a acessibilidade.
    • É uma unidade de medida mais comumente utilizada pela sua capacidade de controle de desenvolvimento, consistencia de tamanhos e promoção de autonomia à pessoa usuária.

  • viewport: é a área visivel de uma página web, podendo variar de dispositivo para dispositivo. É calculada com base no tamanho da área da nossa página que o usuário esta vendo.
    • vw (ou viewport width) é 1% da largura da tela do dispositivo utilizado.
    • wh (ou viewport height) é 1% da altura da tela do dispositivo utilizado.
    • vmin é uma unidade que vai pegar como referencia o menor valor entre largura e altura da viewport de um dispositivo.
    • vmax é uma unidade que vai pegar como referencia o maior valor entre largura e altura da viewport de um dispositivo.

  • ex: toma como referencia a altura do caracter x minusculo da fonte que está sendo utilizada.
    • Muda quando a fonte é alterada.
    • Não é uma unidade de medida muito utilizada.

  • ch: toma como medida a largura do caracter 0 da fonte que está sendo utilizada.
    • Caso seja uma fonte monoespaçada, estamos dizendo que 1 ch cabem 1 caracter.
    • Muda quando a fonte é alterada.
    • Não é uma unidade de medida muito utilizada.

Função cal ( )

Util para realizar calculos de medidas, intercalando valores absolutos com relativos. Realiza-se pela expressão:

propriedade: cal ( valor1 operador valor2), exemplo: width: cal (100% - 30px).
