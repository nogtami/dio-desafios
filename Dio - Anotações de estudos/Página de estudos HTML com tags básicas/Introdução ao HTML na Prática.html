O que é HTML? (Voltar)

HyperText Markup Language (HTML) é uma linguagem de marcação de texto utilizada na construção de página WEB. Como um componente WEB básico, permite a inserção de conteúdo e estabelece uma estrutura básica para um website. Portanto, pode ser considerada um "esqueleto" do website, servido para dar significado e organização para suas informações.

Não é uma linguagem de programação!

Estrutura básica de um documento .html (Voltar)

Este tipo de documento apresenta a seguinte estrutura:

  • Head (ou cabeçalho), onde são fornecidas informações sobre o documento, como títulos da página e metadados.
  • Body (ou corpo do texto), que são os conteúdos do documento que serão lido pelo navegador e apresentados à pessoa usuária.
  • Tags (ou elementos), que são um conjunto de estruturas que se conectam entre si dentro de uma página WEB.São utilizados para informar o navegador o tipo de estrutura que está sendo construida, podendo ser um título, parágrafo, imagens, links, citações, etc.
    • É formada por sinal de menor que (<) + nome do elemento + sinal de maior que (>) = < nomedoelemnto >1
    • OBS: Para fechar, inclui / (barra) após o sinal de menor que (<).
  • Atributos são palavras especiais que funcionam como modificadores de elementos e são utilizados dentro das tags para conferi-lhes algum tipo de comportamento.

Algumas tags e atributos (Voltar)

Para além das tags estruturais apontadas2 acima, temos muitas outras, como:

  • h1 até h6 para hierarquias de títulos;
  • p para parágrafos;
  • ol para listas ordenadas, onde li é uma tag-filha;
  • ul para listas não ordenadas, onde li é uma tag-filha;
  • a para ancoragem/inserção de links no texto;
  • hr para delimitação de conteudos;
  • del para definir um texto que foi deletado de um documento;
  • abbr para definir uma abreviação ou acrônimo.

Criando formulários em HTML (Voltar)

Para a criação de formulários através de HTML, utiliza-se as seguintes tags:3

  • form para iniciar um formulário (funciona como a tag html);
  • input para inserir um campo de preenchimento. Com o atributo type, podem ser inseridos as seguintes informações.
    • text para digitar texto;
    • number para digitar numeros
    • button para criar um botão;
    • range para criar uma escala;
    • password para inserir uma senha;
    • color para selecionar uma cor;
    • e-mail para inserir um endereço de e-mail;
    • url para inserir uma url;
    • date para incluir uma data;
    • file para anexar um arquivo;
    • hidden para esconder um campo
    • search para incluir um campo de busca, porém só funciona no Chrome;
    • checkbox para selecionar muitas opcões;
    • radio para selecionar um única opção.
  • br para quebrar/break linha;
  • label para apresenta uma legenda para um item, como um campo de preenchimento;
  • button para cria um botão de ação;
    • button para criar um botão simples, que poderá ser personalizado depois;
    • reset para limpar os campos que estão dentro do formuláio;
    • submit para enviar o formulário.
  • select para inserir um campo com valores pré-definidos. Precisa de uma tag-filha chamada option para apresentar as opções de seleção pré-definidas ao usuário.
  • textarea para inserir um texto com várias linhas, como mensagens, comentários, etc.

Estruturando o HTML e formatações (Voltar)

Para formatar alguns documentos considerando-se também a acessibilidade, é válido revisar algumas tags e aprender como utilizar outras.

  • Para textos, temos:
    • i para itálico;
    • strong para negrito, enfatizando semanticamente o texto, o que facilita acessibilidade para pessoas com deficiência visual (visualmente é igual ao b);
    • b para negrito simples, sem reforço semantico (visualmente é igual ao strong);
    • u para sublinhado;
    • mark para marcar o texto;
    • sup para elevar um texto;
    • sub para rebaixar um texto;
    • blockquote para delimitar um espaço de citação;
    • small para definir texto menores;
    • font para formatação de tipografia, precisando obrigatoriamente de atributos como face e color para funcionar. Incluir mais de uma opção de fonte tipográfica pode auxiliar na acessibilidade. É uma tag que está caindo em desuso por conta do CSS.
  • div para particulionar horizontalmente o contéudo, auxiliando na arquitetura da página (blocado);
  • span para particiona um contéudo especifico, facilitando sua customização (não-blocado);
  • fieldset para agrupa campos de formulário, criando linhas e legendas ao agrupamento.
  • embed para plugar um conteúdo externo ao seu site. Não é mais utilizada, pois semanticamente perdeu força para outras tags como video.
  • iframe para permite abrir uma janela na sua página com contéudo de outra página, permitindo carregar sites inteiros dentro do seu site.
    Pode afetar a segurança do site. Estudar implementação via JavaScript.

Trabalhando com mídias (Voltar)

Tags de mídia são tags voltadas para inclusão de imagens e/ou sons no seu documento. As mais comuns podem ser vistas abaixo.

  • img para incluir imagens. Se atentar as extensões dos arquivos de imagens, pois isso pode impactar na resolução e no carregamento de páginas. Recomenda-se usar a tag sgv para adicionar arquivos vetoriais com esta mesma extensão. Para aumentar a acessibilidade do seu site, incluia o atributo alt para incluir uma descrição detalhada de todas as imagens.
  • audio para incluir audios. Definições de design de players são nativas do navegador, podendo ser incluidos os controles pelos atributo controls. Precisa da tag-filha source para funcionar. Adicione uma mensagem ao usuário para que ele seja avisado caso o navegador dele não suporte a mídia.
  • video para incluir videos. Mesma situação que a tag audio. Recomenda-se adicionar videos com diferentes extensões para oferecer varias opções de execução ao navegador.
  • track é uma tag auxiliar da tag video e serve para incluir legendas nos seus videos. Tag recomendada para aumentar a acessibilidade do documento.
  • iframe para incorporar páginas externas ao seu site.

Criando tabelas (Voltar)

Tabelas são uma forma de apresentação de dados composta por linhas e colunas. A intersecção de linhas e colunas compõem uma célula. As tags referentes a tabela são:

  • table para incluir uma tabela no seu documento. Precisa de tags auxiliares.
    • tr para incluir linhas na tabela.
    • td para incluir colunas.
    • th para definir um cabeçalho para uma tabela. Facilita na acessibilidade eu endereçar cada informação de acordo com sua localização na tabela.
    • tbody para agrupar células que serão o corpo da tabela. Facilita na organização do documento e no desenvolvimento de uma lógica posterior implementada por uma linguagem de programação.
    • theadpara agrupar células que serão o cabeçalho da tabela. Facilita na organização do documento e no desenvolvimento de uma lógica posterior implementada por uma linguagem de programação.
    • tfootpara agrupar células que serão o corpo da tabela. Facilita na organização do documento e no desenvolvimento de uma lógica posterior implementada por uma linguagem de programação.
    • caption para criar uma descrição para uma tabela que será visível ao usuário. Pode ser lida pelo leitor de tela. É diferente do atributo summary pois este ultimo não fica exposto ao usuário, mas também é lido pelo leitor de tela e pode conter mais informações.
