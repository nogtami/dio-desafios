O que é HTML? (Voltar)

HyperText Markup Language (HTML) é uma linguagem de marcação de texto utilizada na construção de página WEB. Como um componente WEB básico, permite a inserção de conteúdo e estabelece uma estrutura básica para um website. Portanto, pode ser considerada um "esqueleto" do website, servido para dar significado e organização para suas informações.

Não é uma linguagem de programação!

Estrutura básica de um documento .html (Voltar)

Este tipo de documento apresenta a seguinte estrutura:

  • Head (ou cabeçalho), onde são fornecidas informações sobre o documento, como títulos da página e metadados.
  • Body (ou corpo do texto), que são os conteúdos do documento que serão lido pelo navegador e apresentados à pessoa usuária.
  • Tags (ou elementos), que são um conjunto de estruturas que se conectam entre si dentro de uma página WEB.São utilizados para informar o navegador o tipo de estrutura que está sendo construida, podendo ser um título, parágrafo, imagens, links, citações, etc.
    • É formada por sinal de menor que (<) + nome do elemento + sinal de maior que (>) = < nomedoelemnto >*
    • OBS: Para fechar, inclui / (barra) após o sinal de menor que (<).
  • Atributos são palavras especiais que funcionam como modificadores de elementos e são utilizados dentro das tags para conferi-lhes algum tipo de comportamento.

* Retire os espaços para validar as tags.

Algumas tags e atributos (Voltar)

Para além das tags estruturais apontadas** acima, temos muitas outras, como:

  • h1 até h6 para hierarquias de títulos;
  • p para parágrafos;
  • ol + li para listas ordenadas;
  • ul + li para listas não ordenadas;
  • mark para marcação de texto;
  • small para definir texto menores
  • i para italico;
  • u para sublinado;
  • strong para negrito;
  • a para ancoragem/inserção de links no texto;
  • hr para delimitação de conteudos;
  • sub para texto subescritos;
  • sup para textos superescritos;
  • blockquote para destacar citações;
  • del para definir um texto que foi deletado de um documento;
  • abbr para definir uma abreviação ou acrônimo.
